# European FMCG Market Intelligence Project

End-to-end commercial intelligence analysis of the European food retail market, covering France, Netherlands, Germany, Belgium, Sweden, and Denmark (2016–2023).

## Project Overview

This project simulates the full workflow of a Market Intelligence / Commercial Analyst role — from raw data to executive-ready insight. Built using SQL, Excel (Power Query, Power Pivot, DAX), and Power BI on real Eurostat retail trade data.

## Key Findings

- **TAM (Total Addressable Market):** €529B across the 6-country food retail market
- **SAM (Serviceable Addressable Market):** €230B
- **SOM (Serviceable Obtainable Market):** €6.9B (conservative) – €11.5B (optimistic)
- France shows the strongest YoY growth trajectory across the analysis period
- Germany and Netherlands present the largest stable-growth opportunities
